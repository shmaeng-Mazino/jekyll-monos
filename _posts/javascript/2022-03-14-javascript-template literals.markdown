Template literals (작성중)
===


> ? Template literals

자바스크립트에서 문자를 다룰 때 어려웠던 점을 해결하기 위해 나온 문법입니다.

-  문자 중간 엔터키 입력이 가능합니다

```javascript
var 문자 = `안녕하세요
반갑습니다`;

```

- 문자 중간에 변수를 집어넣을 때 편리합니다. 

```javascript
var 이름 = 'MAZINO';
var 문자 = `안녕하세요 ${이름} 입니다`;

```
